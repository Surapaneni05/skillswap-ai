# SkillSwap AI

## Overview

SkillSwap AI is a web platform that allows students to learn new skills by exchanging knowledge with other students. Instead of paying for courses, users can teach a skill they know and learn a skill they are interested in from peers.

The platform helps students connect, collaborate, and grow through a community-driven skill exchange system.

## Features

* User Registration and Login
* User Profile Management
* Skills Offered and Skills Wanted
* Smart Skill Matching
* Session Request and Management
* Credit-Based Learning System
* Ratings and Reviews
* AI Career Roadmap Generation
* Real-Time Chat
* Admin Dashboard

## Objective

The main objective of SkillSwap AI is to create a collaborative learning environment where students can teach, learn, and improve their skills through mutual knowledge exchange.
